# re-agent Manual Test Plan

Tests run in `/tmp/re-agent-test`. Each scenario creates a fresh git repo, runs init and/or the hook, then asserts expected state. Clean up between scenarios.

**Agent instructions:** Run each scenario sequentially. For each assertion, print PASS/FAIL. Report a summary at the end.

---

## Setup / Teardown

```bash
# Before each scenario
rm -rf /tmp/re-agent-test && mkdir /tmp/re-agent-test && cd /tmp/re-agent-test
git init
echo '{"name":"test","version":"1.0.0"}' > package.json

# After each scenario
rm -rf /tmp/re-agent-test
```

The CLI entry point is at the repo root: `node <repo>/bin/re-agent.js init`

The hook script is: `bash scripts/setup-agent-symlinks.sh`

---
